Create four functions that will convert from or to dollars. Each function will require one argument: the amount to be converted. It will return the converted amount.

For example:
- dollarsToBitcoin
- bitcoinToDollars
- dollarsToEuros
- eurosToDollars

A second example:
- dollarsToCanadian
- canadianToDollars
- dollarsToPesos
- pesosToDollars

A third example:
- dollarsToYen
- yenToDollars
- dollarsToRupees
- rupeesToDollars
